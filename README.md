# pbip-model-migrator
1. 🔥 Problem

Power BI model migrations require manual updates to tables and column mappings, which is time-consuming and error-prone—especially across large semantic models.

2. ⚡ Solution

This tool automates the refactoring of PBIP files by dynamically mapping tables and columns between source and target models.

3. 🧠 Key Features
Table-level remapping
Column-level mapping
Bulk refactoring across semantic model files
Works directly with PBIP structure
Reduces migration time from hours → minutes

4. 🛠️ Tech Stack
Python
JSON manipulation (PBIP structure)
Power BI semantic models
